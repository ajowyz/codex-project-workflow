# Decisions

## ADR-001: Export format

- 治理状态：已确认
- 证据状态：已验证
- 替代关系：无
- 决定内容：Keep the public export format unchanged.
- 原因：External consumers depend on it.

## ADR-002: Retention period

- 治理状态：已确认
- 证据状态：独立复核
- 替代关系：无
- 决定内容：Retain completed records for 30 days.
- 原因：Balance recovery needs and storage cost.
